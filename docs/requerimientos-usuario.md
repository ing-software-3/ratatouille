# REQ-USUARIO

El usuario podrá registrarse en la plataforma mediante correo y contraseña.
El usuario podrá iniciar sesión para acceder al sistema.
El usuario podrá visualizar el menú de comidas rápidas disponibles.
El usuario podrá buscar productos por nombre o categoría.
El usuario podrá agregar productos al carrito de compras.
El usuario podrá realizar pedidos en línea.
El usuario podrá editar su información personal.
El usuario podrá visualizar el precio total de su pedido.
El usuario podrá guardar productos como favoritos.
El usuario podrá elegir si desea el producto dependiendo la hora de entrega.

# REQ-ADMINISTRADOR 

El administrador podrá agregar nuevos productos al menú.
El administrador podrá editar información de los productos.
El administrador podrá eliminar productos del sistema.

# REQ-SISTEMA 

El sistema deberá almacenar el historial de pedidos del usuario.
El sistema permitirá visualizar el estado del pedido.
El sistema podrá mostrar restaurantes cercanos según la ubicación del usuario.
El usuario podrá cerrar sesión de forma segura.
